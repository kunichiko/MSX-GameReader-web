MSX Game Reader — WinUSB ドライバ インストールガイドWinUSB Driver Install Guide

Windows で MSX Game Reader をブラウザ／libusb から使うための手順 How to make MSX Game Reader accessible from browser / libusb on Windows

なぜ WinUSB ドライバが必要？

MSX Game Reader は USB のベンダクラスデバイス（クラスコード 0xFF）で、Windows 標準のクラスドライバとは関連付けられません。 公式の専用ドライバ（Jungo WinDriver 系）は 32bit Windows XP 専用で、現在の 64bit Windows 10/11 では事実上使えません。

その代わり、Microsoft 製の汎用ドライバ WinUSB.sys を割り当てれば、ブラウザの WebUSB API や libusb 系のツールから直接アクセスできるようになります。 Zadig などの外部ツールは不要で、Windows 標準のデバイスマネージャーだけで完結します。

この手順は Microsoft の 公式情報 に基づきます。Mimic X 用の同種手順（こちら）と同じ流れですが、Game Reader には BOOT モードのような時間制約はありません。

WinUSB の割り当て手順

  1. MSX Game Reader を USB で PC に接続します。
    Windows がデバイスを認識するまで数秒待ってください。
  2. スタートメニューから「デバイス マネージャー」を起動します。
  3. 「ほかのデバイス」（または「ユニバーサル シリアル バス デバイス」）の下に、不明なデバイスとして MSX Game Reader が表示されます。
  4. 右クリックして「プロパティ」を開き、「詳細」タブで「ハードウェア ID」を選び、 USB\VID_1125&PID_AC01 が表示されていることを確認してください。 （これが MSX Game Reader です）
  5. 「ドライバー」タブを開き、「ドライバーの更新」ボタンを押します。
  6. 「コンピューターを参照してドライバーを検索」を選びます。
  7. 「コンピューター上の利用可能なドライバーの一覧から選択します」を選びます。
  8. 「すべてのデバイスを表示」を選んで「次へ」を押し、表示される一覧から 「ユニバーサル シリアル バス デバイス」を選択して「次へ」を押します。
  9. 「製造元」のリストから「WinUsb デバイス」、「モデル」から「WinUsb デバイス」を選び、「次へ」を押します。
  10. セキュリティの警告が出たら「はい」（インストールを続行）を押します。
  11. インストール完了後、デバイスマネージャーで「ユニバーサル シリアル バス デバイス」の下に新しい項目（WinUsb デバイス）として表示されていれば成功です。

動作確認

Web Dumper を Chrome / Edge / Opera で開き、「Connect」ボタンを押してください。 デバイス選択ダイアログに「不明なデバイス (1125:AC01)」のような項目が現れれば、ドライバの割り当ては成功しています。

うまくいかない場合

  • デバイスマネージャーに何も出てこない：
    USB ケーブルや別のポートに変えて試してください。MSX Game Reader 本体の LED が光らない場合は電源が来ていません。
  • 「WinUsb デバイス」が一覧に出ない：
    Windows のバージョンが古い可能性があります。Windows 10 1709 以降、または Windows 11 を推奨します。
  • ドライバを元に戻したい：
    デバイスマネージャーで該当デバイスを右クリック → 「デバイスのアンインストール」→「このデバイスのドライバーを削除」にチェックを入れて実行。 その後、USB を抜き差しすれば「不明なデバイス」状態に戻ります。

Why do I need a WinUSB driver?

MSX Game Reader is a USB vendor-class device (class code 0xFF) and is not associated with any standard Windows class driver. The official vendor driver (based on Jungo WinDriver) is for 32-bit Windows XP only and is effectively unusable on current 64-bit Windows 10/11.

Instead, by binding Microsoft's generic WinUSB.sys driver, you can access the device directly from browsers via the WebUSB API or from libusb-based tools. No external tools such as Zadig are required; everything can be done through Windows Device Manager.

This procedure is based on Microsoft's official documentation. It follows the same steps as the Mimic X guide (here), but Game Reader has no time constraint like BOOT mode.

Steps to assign WinUSB

  1. Connect MSX Game Reader to your PC via USB.
    Wait a few seconds for Windows to detect the device.
  2. Open Device Manager from the Start menu.
  3. Under "Other devices" (or "Universal Serial Bus devices"), an Unknown device entry will appear for the MSX Game Reader.
  4. Right-click → Properties. Switch to the "Details" tab, select "Hardware Ids", and verify that USB\VID_1125&PID_AC01 is shown. (This identifies the MSX Game Reader.)
  5. Switch to the "Driver" tab and click "Update Driver".
  6. Choose "Browse my computer for drivers".
  7. Choose "Let me pick from a list of available drivers on my computer".
  8. Select "Show all devices" and click Next. From the list, choose "Universal Serial Bus devices" and click Next.
  9. Choose "WinUsb Device" as the Manufacturer and "WinUsb Device" as the Model. Click Next.
  10. If a security warning appears, click "Yes" to proceed with the installation.
  11. After installation, the device will appear as a new entry (WinUsb Device) under "Universal Serial Bus devices" in Device Manager. Success!

Verification

Open the Web Dumper in Chrome / Edge / Opera and click the "Connect" button. If "Unknown device (1125:AC01)" or similar appears in the device selection dialog, the driver binding is successful.

Troubleshooting

  • Nothing shows up in Device Manager:
    Try a different USB cable or port. If the LED on the MSX Game Reader doesn't light up, the device isn't getting power.
  • "WinUsb Device" is not in the list:
    Your Windows version may be too old. Windows 10 1709 or later, or Windows 11, is recommended.
  • Reverting the driver:
    Right-click the device in Device Manager → "Uninstall device" → check "Delete the driver software for this device" → OK. Then replug the USB to return to the "Unknown device" state.
